# 🎓 Alumni Management System

A web-based platform that helps colleges and universities connect alumni, students, and administrators. The system enables alumni registration, profile management, networking, event announcements, and communication between alumni and the institution.

## 🚀 Features

### 👨‍🎓 Alumni
- Register and create profile
- Update personal and professional details
- View alumni directory
- Connect with other alumni
- Receive event notifications

### 👩‍💼 Admin
- Manage alumni records
- Verify alumni accounts
- Post announcements and events
- Monitor platform activity

### 📅 Events
- Create and manage alumni events
- View upcoming events
- Event registration and participation

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/alumni-management-system.git
```

### Navigate to Project

```bash
cd alumni-management-system
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm start
```

## 🔑 Environment Variables

Create a `.env` file in the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
